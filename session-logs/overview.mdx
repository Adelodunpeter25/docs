---
title: Session Logs
description: Capture and review terminal session logs
---

Session logging helps with audits, debugging, and post-incident review.

## Capabilities

- Start/stop terminal logging
- View saved logs in the app
- Open logs in dedicated log tabs
- Delete old logs

## Recommended Workflow

1. Start logging before sensitive operations
2. Perform terminal tasks
3. Stop logging when done
4. Review logs for validation and sharing

## Troubleshooting

### Log File Missing or Empty

- Ensure logging was started before command execution.
- Confirm session remained active during logging window.
