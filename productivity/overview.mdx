## Keyboard Shortcuts

Use these core shortcuts daily:

- `Cmd/Ctrl + 1`: Go to Home
- `Cmd/Ctrl + 2`: Go to SFTP
- `Cmd/Ctrl + 3-9`: Switch session tabs
- `Cmd/Ctrl + T`: New connection
- `Cmd/Ctrl + L`: New local terminal
- `Cmd/Ctrl + W`: Close current tab
- `Cmd/Ctrl + ,`: Open settings
- `Cmd/Ctrl + Shift + /`: Show keyboard shortcuts
- `Cmd/Ctrl + S`: Toggle terminal sidebar
- `Cmd/Ctrl + K`: Clear terminal
- `Cmd/Ctrl + F`: Search in terminal
- `Cmd/Ctrl + R`: Refresh SFTP file lists

## Command Palette

Open command palette with `Cmd/Ctrl + P`.

Current actions include:

- New connection
- New local terminal
- New workspace tab
- Open settings
- Open keyboard shortcuts
- Open export/import
