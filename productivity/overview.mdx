## Keyboard Shortcuts

Keyboard shortcuts are configurable in-app from **Keyboard Shortcuts** settings.

Core defaults:
- `Cmd/Ctrl + 1`: Home
- `Cmd/Ctrl + 2`: SFTP
- `Cmd/Ctrl + 3-9`: Session tabs
- `Cmd/Ctrl + P`: Command palette
- `Cmd/Ctrl + T`: New connection
- `Cmd/Ctrl + L`: New local terminal

## Command Palette

Open with `Cmd/Ctrl + P` and run actions like:
- New connection
- New local terminal
- Open settings
- Open keyboard shortcuts
- Open export/import
