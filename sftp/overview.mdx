# SFTP File Transfers

FreeSH includes a dual-pane SFTP interface for local and remote file operations.

## Core Capabilities

- Browse local and remote directories
- Upload and download files/folders
- Bulk operations for multiple items
- Remote-to-remote transfer support
- File preview for text-based files

## Opening SFTP

From an active remote session tab:

1. Open tab context menu
2. Select **Open in SFTP**

You can also open SFTP from the main navigation.

## Navigation

- Double-click folders to enter
- Use path input to jump paths
- Use remote path autocomplete where available

## File Operations

- Create folder
- Rename
- Delete
- Change permissions
- Transfer with progress feedback

## File Preview

Text-based files can be previewed and edited directly in-app.

## Troubleshooting

### Cannot Browse Remote Path

- Confirm the SSH session is active.
- Verify account permissions for the target directory.

### Transfer Fails

- Check remote disk space and permissions.
- Retry after reconnecting if session was interrupted.
