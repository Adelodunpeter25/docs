---
title: What is FreeSH
description: A free, open-source, local-first SSH client.
---

FreeSH combines the functionality of a terminal emulator, SSH client, and SFTP file manager into a single application. Whether you're managing multiple servers, transferring files between remote locations, or executing commands across your infrastructure, FreeSH provides the tools you need.

## Key Features

### Terminal Management
- **Multiple Sessions**: Open and manage multiple SSH connections simultaneously with tabbed interface
- **Local Terminal**: Built-in local terminal for quick command execution

### Secure Authentication
- **Multiple Auth Methods**: Support for password and SSH key authentication
- **SSH Key Management**: Browse and select SSH keys with built-in file picker
- **System Keychain Integration**: Securely store credentials using your system's keychain
- **Known Hosts Management**: Review and confirm host keys before adding to trusted hosts

### Advanced SFTP
- **Dual-Panel Interface**: Side-by-side file browsing for easy transfers
- **Remote-to-Remote Transfers**: Transfer files directly between servers without downloading locally
- **File Preview**: View and edit remote files directly in the application

### Port Forwarding
- **Local Forwarding**: Forward local ports to remote destinations
- **Remote Forwarding**: Forward remote ports to local destinations
- **Dynamic Forwarding**: SOCKS proxy support for secure browsing

### Productivity Features
- **Command Snippets**: Save and organize frequently used commands
- **Command History**: Track and reuse commands across sessions
- **Variables & Placeholders**: Create dynamic snippets with runtime input
- **Connection Groups**: Organize connections into folders and groups
- **Session Logging**: Save terminal sessions to file for later review
- **Tab Management**: Pin important sessions and rename tabs for easy identification
- **Keyboard Shortcuts**: Speed up your workflow with comprehensive keyboard shortcuts

### Security & Privacy
- **Local-First Architecture**: All data stored locally on your machine, no cloud sync
- **Encrypted Credentials**: Passwords and keys secured with system keychain
- **No Telemetry**: Zero tracking or data collection
- **Offline Capable**: Full functionality without internet (except remote connections)

## Open Source

FreeSH is open source software. Contributions, bug reports, and feature requests are welcome on our [GitHub repository](https://github.com/Adelodunpeter25/freessh).
